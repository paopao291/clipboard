# 📷 シール帳 - 画像コレクションアプリ

画像をコピー＆ペーストして、自由に配置・編集できるシール帳のようなWebアプリです。

## ✨ 機能

### 画像の追加方法

#### PC
- **キーボードペースト**: 画像をコピーして `Ctrl+V` / `Cmd+V`
- **ギャラリーボタン**: ファイル選択から複数画像を追加

#### スマホ
- **長押しペースト**: 画像をコピーして画面を長押し→「ペースト」を選択
- **📁 ギャラリーボタン**: カメラロールから画像を選択
- **📸 カメラボタン**: カメラで直接撮影して追加

### 画像の操作

#### PC
- **移動**: 画像をドラッグ
- **拡大縮小**: 画像の上でマウスホイールをスクロール
- **回転**: `Shift` キーを押しながらドラッグ
- **削除**: 画像にカーソルを合わせて×ボタンをクリック

#### スマホ
- **移動**: 1本指でドラッグ
- **拡大縮小・回転**: 2本指でピンチ操作
- **削除**: 画像をタップして×ボタンをタップ

### その他の機能
- **💾 保存**: コレクションをブラウザに保存
- **📂 読み込み**: 保存したコレクションを読み込み
- **🗑️ すべてクリア**: すべての画像を削除
- **自動保存**: ページを閉じる時に自動保存

## 🚀 起動方法

```bash
# Pythonサーバーを使用する場合
python3 main.py
# http://localhost:5000 にアクセス

# または、シンプルなHTTPサーバー
cd public && python3 -m http.server 8000
# http://localhost:8000 にアクセス
```

## 📱 スマホでの使い方のコツ

1. 他のアプリで画像をコピー（例：写真アプリで画像を長押し→「コピー」）
2. このアプリに戻る
3. 画面の空いているところを長押し
4. 表示されたメニューから「ペースト」を選択
5. 画像が中央に追加されます！

## 💡 ヒント

- 複数の画像を同時にギャラリーから選択できます
- 画像は自動的にブラウザに保存されます（LocalStorage使用）
- スマホでは2本指で同時に拡大縮小と回転ができます

## 🛠 技術スタック

- **HTML5**: 構造
- **CSS3**: モダンなUIデザイン、レスポンシブ対応
- **JavaScript (Vanilla)**: Clipboard API、タッチイベント、LocalStorage
- **Flask**: 静的ファイル配信（オプション）

## 📝 ブラウザ対応

- Chrome, Edge, Firefox, Safari (最新版)
- iOS Safari, Chrome Mobile, Samsung Internet

## ⚠️ 注意事項

- スマホのブラウザによって、クリップボードへのアクセス制限が異なる場合があります
- iOSでは、画像のペースト機能に一部制限があります
- 大量の画像を保存すると、LocalStorageの容量制限に達する可能性があります
